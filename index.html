RenderTarget Optimization

Overview

Integrated and configured one of the most technically complex bonus game systems consisting of more than 1,300 objects. The project combines Spine and After Effects animations into a single production-ready game feature. Each object with an assigned instance name contains its own Event, allowing gameplay logic to be driven directly from animation timelines. When an animation reaches a specific frame, the Event automatically triggers the required state in another object, ensuring precise synchronization between animations and gameplay.

Performance Optimization

To optimize video resources, the project extensively utilized RenderTarget. Animated content was rendered once, cached, and reused efficiently, reducing GPU load and video memory usage while maintaining visual quality.
